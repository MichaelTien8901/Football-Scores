## football-data.org API KEY

Need to add a file /app/src/main/res/values/security_strings.xml

```
<?xml version="1.0" encoding="utf-8"?>
<resources>
    <string name="api_key" translatable="false">[football-data.org API KEY HERE]</string>
</resources>
```
